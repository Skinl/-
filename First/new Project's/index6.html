CSS селекторы

CSS селекторы Пример Описание
.class .myClass Выбирает все элементы с классом myClass(class="myClass")
#id #main Выбирает элемент с индефикатором main(id="main)"
* * Выбирает все элементы
Элемент div Выбор всех элементов <div>
элемент, элемент div, span Выбор всех элементов <div> <span>
[атрибут] [href] Все элементы с атрибутом href
[атрибут="значение"] [href="https://vk.com/"] Выбирает все элементы с атрибутом href, значение которого ТОЧНО совпадает со значением указанным в селекторе
[атрибут~="значение"] href~="https://vk.com" Выбирает все элементы с атрибутов href, в значении которого(в любом месте)встречается подстрока(в виде отдельного слова) "vk.com"
[атрибут~="значение"] [href*="https:vk.com"] Выбирает все элементы с атрибутов href, в значении которого(в любом месте)встречается подстрока(в виде отдельного слова или его части) "vk.com"
Элемент элемент div span Выбор всех элементов <span> внутри <div>
элемент+элемент div+p Выбирает все элементы <p>, расположенные сразу после элементов <div>
элемент~элемент p~div Выбирает все элементы <div>, расположенные сразу перед элементом <p>
элемент: nth-child(номер) div:nth-child(3) Выбирает третий элемент,который лежит внутри <div>
